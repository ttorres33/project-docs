Update all project documentation by running these commands sequentially:

1. First, run `/project-docs:process-notes` to capture the current session
2. Then, run `/project-docs:readme` to update README.md
3. Finally, run `/project-docs:architecture` to update ARCHITECTURE.md

Wait for each command to complete before starting the next one.
